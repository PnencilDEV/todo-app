## ðŸŽ¯ How to Use

### Adding Tasks
1. Click the input field or press `Ctrl + /`
2. Type your task (up to 200 characters)
3. Press `Enter` or click "Add Task"

### Managing Tasks
- **Complete**: Click the circle checkbox next to any task
- **Edit**: Click the yellow edit button (pencil icon)
- **Delete**: Click the red delete button (trash icon)

### Filtering
- **All Tasks**: View everything
- **Pending**: See only incomplete tasks
- **Completed**: View finished tasks

### Keyboard Shortcuts
- `Ctrl + /`: Focus on task input
- `Enter`: Add new task (when input is focused)
- `Enter`: Save task edit (when editing)
- `Escape`: Close modals
